# Image_classification
Use the pytorch tool to train the classification model provided in the tool and compare the relevant performance.


1. `doc`文件夹

   - 保存训练结果；

2. `models`文件夹

   - 各模型模型设定文件、训练文件；
   - 模型：DenseNet、EfficientNet、MobileNetV3、RegNet、ResNet模型文件；
     - 原始模型，完全没有微调的模型；
     - 微调分类器模型；

3. `utils`文件夹

   - 各种处理的小工具；
